How deliberate works

1
You ask a question. Deliberate sends it simultaneously to multiple AI models — currently Groq, Gemini, and Mixtral 8x7B (via HuggingFace), with support for custom models.
2
The models deliberate. Each model responds independently. The engine measures how much they agree or disagree using semantic distance.
3
A tiebreaker resolves conflicts. When the first two models disagree significantly, a third model acts as tiebreaker — siding with whichever answer it agrees with more.
4
You get a filtered answer — with a confidence score and an optional transparency view showing exactly what each model said and how much they disagreed.
Why does this matter? Any single AI model can confidently produce a wrong answer. When multiple independent models agree, that convergence is meaningful signal. When they disagree, that's equally important — and you should know about it.

api keys

how tiers work
Tier 1 models (Groq, Gemini) respond in parallel to every question. If they broadly agree, you get their answer immediately.

Tier 2 (Mixtral 8x7B) only activates when T1 models disagree significantly. It acts as a tiebreaker — siding with whichever T1 answer it agrees with more. Without a T2 key, deliberate still works but cannot resolve strong disagreements.
add a custom model
Add any OpenAI-compatible model to the pool. Groq and HuggingFace both use this format. Set tier to 1 for primary models or 2 for tiebreakers.
